Set up Multiple Github Users
@ · Jan 1, 0001 · 1 min read · Update at Jan 1, 0001

Add keys to different users, set up different rsa files ssh-keygen -t rsa -b 4096 -C “your_email@example.com“ Enter a file in which to save the key (/home/.ssh/id_rsa): ~/.ssh/user1_id_rsa After set up, go to config: vim ~/.ssh/config

host vimana511.github.com
    Hostname github.com
    User vimana511
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/v_id_rsa
    IdentitiesOnly yes

host github.com
    Hostname user2.github.com
    User user2 
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/id_rsa
    IdentitiesOnly yes

In your other repo, set git to other users: git config user.name “user1” git config user.email user@nili.ca

Check the set up: vim .git/config

ssh -T git@vimana511.github.com ssh -T git@user2.github.com
